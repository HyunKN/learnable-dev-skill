# Template - Tech Decision Study Note

Leave a short note after a meaningful technical decision so the team can remember both the decision and what was learned from it.

- Which files or layers use it?
- What alternatives were considered?

- What is it?
- Why was it used here?
- How is it used in this project?
- When would we avoid it?
