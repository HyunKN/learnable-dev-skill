# Language and Output

## Language Rule

When the user asks for a reusable document, template, or playbook and has not already said Korean or English, ask first which language they want.

Good first question:

- Do you want the document in Korean or English?

## Output Preference

Prefer outputs that can be reused immediately:

- markdown documents
- checklists
- templates
- operating guides
- short principle docs

Avoid leaving the user with only loose chat advice when a reusable file would be more helpful.
